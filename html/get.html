Conceptos generales de

jQuery GET

¿Qué es?

jQuery GET es una función proporcionada por la biblioteca de JavaScript jQuery, que se utiliza para realizar solicitudes HTTP GET (obtener) asíncronas a un servidor web. Permite obtener datos de un servidor y actualizar el contenido de una página web sin tener que recargarla por completo.

Características

1. Utiliza AJAX (Asynchronous JavaScript and XML) para realizar solicitudes asíncronas al servidor.
2. Puede realizar acciones como actualizar el contenido de una página web, mostrar los datos en un cuadro de diálogo, modificar el estilo de los elementos de la página, etc.
3. Puede procesar los datos recibidos en diferentes formatos, como JSON, XML o texto plano.

Estructura

1. url La URL del servidor a la cual se enviará la solicitud GET.
2. data (opcional): Los datos que se enviarán al servidor junto con la solicitud GET.
3. success La función de devolución de llamada que se ejecutará si la solicitud es exitosa.
4. dataType (opcional): El tipo de datos esperados en la respuesta del servidor. Puede ser "xml", "json", "script", "html" o "text".

APLICACIÓN

Obtener contenido: texto (), html () y val ()

Tres métodos jQuery simples, pero útiles, para la manipulación de DOM son:

text()- Establece o devuelve el contenido de texto de los elementos seleccionados

html()- Establece o devuelve el contenido de los elementos seleccionados (incluido el marcado HTML)

val()- Establece o devuelve el valor de los campos de formulario

El siguiente ejemplo demuestra cómo obtener contenido con los métodos jQuery text() y html():

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("#btn1").click(function(){
            alert("Text: " + $("#test").text());
        });
        $("#btn2").click(function(){
            alert("HTML: " + $("#test").html());
        });
    });
    </script>
    </head>
    <body>

    <p id="test">This is some <b>bold</b> text in a paragraph.p>

    <button id="btn1">Show Text</button>
    <button id="btn2">Show HTML</button>

    </body>
    </html>

El siguiente ejemplo demuestra cómo obtener el valor de un campo de entrada con el método val():

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("button").click(function(){
            alert("Value: " + $("#test").val());
        });
    });
    </script>
    </head>
    <body>

    <p>Name: <input type="text" id="test" value="Mickey Mouse"></p>

    <button>Show Value</button>

    </body>
    </html>

Obtener atributos - attr()

El attr() mpetodo se utiliza para obtener valores de atributos.

El siguiente ejemplo demuestra cómo obtener el valor del atributo href en un enlace:

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("button").click(function(){
            alert($("#w3s").attr("href"));
        });
    });
    </script>
    </head>
    <body>

    <p><a href="https://www.w3schools.com" id="w3s">W3Schools.com</a></p>

    <button>Show href Value</button>

    </body>
    </html>
