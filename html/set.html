Conceptos generales de

jQuery SET

¿Qué es?

jQuery proporciona una función llamada "jQuery .set()" que se utiliza para establecer propiedades CSS en uno o más elementos seleccionados.

Características

1. Puede modificar propiedades como el color de fondo, el tamaño de fuente, el margen, el relleno, la opacidad, etc.
2. Puede usar selectores de jQuery para seleccionar uno o más elementos en los que se desea establecer las propiedades CSS. Pueden ser elementos específicos, clases, ID o cualquier otro selector válido de jQuery.
3. El valor que se asigna a una propiedad CSS puede ser un valor fijo o un valor dinámico basado en variables o cálculos.

Estructura

1. selector: La URL del servidor a la cual se enviará la solicitud GET.
2. propertyName: Los datos que se enviarán al servidor junto con la solicitud GET.
3. value: La función de devolución de llamada que se ejecutará si la solicitud es exitosa.

APLICACIÓN

Establecer contenido: texto (), html () y val ()

Usaremos los mismos tres métodos de la página anterior para configurar el contenido :

text()- Establece o devuelve el contenido de texto de los elementos seleccionados

html()- Establece o devuelve el contenido de los elementos seleccionados (incluido el marcado HTML)

val()- Establece o devuelve el valor de los campos de formulario

El siguiente ejemplo demuestra cómo obtener contenido con los métodos jQuery text(), html() y val()

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("#btn1").click(function(){
            $("#test1").text("Hello world!");
        });
        $("#btn2").click(function(){
            $("#test2").html("<b>Hello world!</b>");
        });
        $("#btn3").click(function(){
            $("#test3").val("Dolly Duck");
        });
    });
    </script>
    </head>
    <body>

    <p id="test1">This is a paragraph.</p>
    <p id="test2">This is another paragraph.</p>

    <p>Input field: <input type="text" id="test3"span >value="Mickey Mouse"></p>

    <button id="btn1">Set Text</button>
    <button id="btn2">Set HTML</button>
    <button id="btn3">Set Value</button>

    </body>
    </html>

Una función de devolución de llamada para text(), html() y val()

Los tres métodos jQuery anteriores: text(), html() y val(), también vienen con una función de devolución de llamada. La función de devolución de llamada tiene dos parámetros: el índice del elemento actual en la lista de elementos seleccionados y el valor original (antiguo). Luego devuelve la cadena que desea usar como el nuevo valor de la función

El siguiente ejemplo demuestra text() y html() con una función de devolución de llamada:

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("button").click(function(){
            alert("Value: " + $("#test").val());
        });
    });
    </script>
    </head>
    <body>

    <p>Name: <input type="text" id="test" value="Mickey Mouse"></p>

    <button>Show Value</button>

    </body>
    </html>

Establecer atributos - attr()

El método attr() también se usa para establecer/cambiar valores de atributos.

El siguiente ejemplo demuestra cómo obtener el valor del atributo href en un enlace:

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("button").click(function(){
            $("#w3s").attr("href", "https://www.w3schools.com/jquery/");
        });
    });
    </script>
    </head>
    <body>

    <p><a href="https://www.w3schools.com" id="w3s">W3Schools.com</a></p>

    <button>Change href Value</button>

    <p>Mouse over the link (or click on it) to see that the value of the href attribute has changed.</p>

    </body>
    </html>
