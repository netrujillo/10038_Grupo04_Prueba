Conceptos generales de

jQuery ADD

¿Qué es?

jQuery ADD proporciona métodos y funciones relacionadas que permiten agregar elementos, clases o contenido a una selección existente.

Características

1. Puede modificar propiedades como el color de fondo, el tamaño de fuente, el margen, el relleno, la opacidad, etc.
2. Puede usar selectores de jQuery para seleccionar uno o más elementos en los que se desea establecer las propiedades CSS. Pueden ser elementos específicos, clases, ID o cualquier otro selector válido de jQuery.
3. El valor que se asigna a una propiedad CSS puede ser un valor fijo o un valor dinámico basado en variables o cálculos.

Estructura

1. .append(): La URL del servidor a la cual se enviará la solicitud GET.
2. .prepend(): Los datos que se enviarán al servidor junto con la solicitud GET.
3. after(): La función de devolución de llamada que se ejecutará si la solicitud es exitosa.
4. before(): La función de devolución de llamada que se ejecutará si la solicitud es exitosa.

APLICACIÓN

Agregar nuevo contenido HTML

Veremos cuatro métodos jQuery que se utilizan para agregar contenido nuevo:

append() - Inserta contenido al final de los elementos seleccionados

prepend() - Inserta contenido al principio de los elementos seleccionados

after() - Inserta contenido después de los elementos seleccionados

before() - Inserta contenido antes de los elementos seleccionados

Método jQuery append()

El método jQuery append() inserta contenido AL FINAL de los elementos HTML seleccionados.

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("#btn1").click(function(){
            $("p").append(" <b>Appended text</b>.");
        });

        $("#btn2").click(function(){
            $("ol").append("<li>Appended item</li>");
        });
    });
    </script>
    </head>
    <body>
    <p>This is a paragraph.</p>
    <p>This is another paragraph.</p>

    <ol>
        <li>List item 1</li>
        <li>List item 2</li>
        <li>List item 3</li>
    </ol>

    <button id="btn1">Append text</button>
    <button id="btn2">Append list items</button>

    </body>
    </html>

Método jQuery prepend()

El prepend() método jQuery inserta contenido AL PRINCIPIO de los elementos HTML seleccionados.

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("#btn1").click(function(){
            $("p").prepend("<b>Prepended text</b>. ");
        });
        $("#btn2").click(function(){
            $("ol").prepend("<li>Prepended item</li>");
        });
    });
    </script>
    </head>
    <body>
    <p>This is a paragraph.</p>
    <p>This is another paragraph.</p>

    <ol>
      <li>List item 1</li>
      <li>List item 2</li>
      <li>List item 3</li>
    </ol>

    <button id="btn1">Prepend text</button>
    <button id="btn2">Prepend list item</button>

    </body>
    </html>

Métodos jQuery after() y before()

El método after() inserta contenido DESPUÉS de los elementos HTML seleccionados.

El método before() inserta contenido ANTES de los elementos HTML seleccionados.

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("#btn1").click(function(){
            $("img").before("<b>Before</b>");
        });

        $("#btn2").click(function(){
            $("img").after("<i>After</i>");
        });
    });
    </script>
    </head>
    <body>
    <img src="/images/w3jquery.gif" alt="jQuery" width="100" height="140"><br><br>

    <button id="btn1">Insert before</button>
    <button id="btn2">Insert after</button>

    </body>
    </html>
