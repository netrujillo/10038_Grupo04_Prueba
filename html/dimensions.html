Conceptos generales de

jQuery Dimensions

¿Qué es?

Es un conjunto de funciones y métodos proporcionados por la biblioteca jQuery que permiten acceder y manipular las dimensiones (tamaño y posición) de elementos HTML seleccionados. Estas funciones facilitan la obtención de información sobre el tamaño y la posición de los elementos, así como la modificación dinámica de estas dimensiones.

Características

1. permiten acceder y manipular las dimensiones (tamaño y posición) de elementos HTML seleccionados de manera conveniente y eficiente.
2. Permiten acceder y manipular las dimensiones (tamaño y posición) de elementos HTML seleccionados de manera conveniente y eficiente.

Estructura

1. width(): Devuelve el ancho calculado del primer elemento seleccionado.
2. height(): Devuelve la altura calculada del primer elemento seleccionado.
3. position(): Devuelve un objeto con las coordenadas de posición (izquierda y superior) del primer elemento seleccionado en relación con su padre posicionado más cercano.

Métodos jQuery width() y height()

El width() método establece o devuelve el ancho de un elemento (excluye relleno, borde y margen).

El height() método establece o devuelve la altura de un elemento (excluye relleno, borde y margen).

El siguiente ejemplo devuelve el ancho y alto de un <div> elemento especificado:

                                    
                                    
    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"></script>
    <script>
    $(document).ready(function(){
        $("button").click(function(){
            var txt = "";
            txt += "Width of div: " + $("#div1").width() + "</br>";
            txt += "Height of div: " + $("#div1").height();
            $("#div1").html(txt);
        });
    });
    </script>
    <style>
    #div1 {
        height: 100px;
        width: 300px;
        padding: 10px;
        margin: 3px;
        border: 1px solid blue;
        background-color: lightblue;
    }
    </style>
    </head>
    <body>

    <div id="div1"></div>
    <br>

    <button>Display dimensions of div</button>

    <p>width() - returns the width of an element.</p>
    <p>height() - returns the height of an element.</p>

    </body>
    </html>
